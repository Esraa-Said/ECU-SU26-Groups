# 🎯 JavaScript Mini Projects 


## 🧾 Project 1: Smart University Exam Processing System

Build a system that simulates how a university processes exam results for multiple students in sequence.

The system should evaluate each student one by one, check their attendance, exam score, and special conditions like absence or system errors.

It should decide whether each student passes, fails, or is not allowed to take the exam based on predefined rules.

The system should also be able to stop completely in case of a critical error.

---

## 🧾 Project 2: Online Store Order Management System

Create a simulation for an online store that processes customer orders one after another.

Each order must be validated before processing, including checking availability, order status, and special cases such as invalid or cancelled orders.

The system should be able to skip invalid orders while continuing normal processing for valid ones.

In extreme cases like system overload or stock failure, the system should stop processing further orders.

---

## 🧾 Project 3: Hospital Emergency Triage System

Build a system that simulates a hospital emergency room workflow.

Patients arrive in sequence and each patient is evaluated based on severity, symptoms, and urgency level.

The system should assign priority levels and decide the order of treatment.

It should also handle special cases like missing patient data or emergency critical conditions that require immediate attention.

---

## 🧾 Project 4: Delivery Tracking and Logistics System

Create a system that simulates a delivery company tracking multiple shipments over time.

Each shipment goes through multiple stages such as processing, shipping, delay, and delivery completion.

The system should evaluate the status of each shipment and decide the next action accordingly.

It should also handle exceptions like failed deliveries, lost packages, or system interruptions.

---

## 🧾 Project 5: Smart Traffic Control Simulation System

Build a simulation of a smart traffic control system that manages traffic flow based on changing signals.

The system should continuously evaluate traffic light states and decide actions for vehicles accordingly.

It should handle different conditions such as congestion, accidents, and emergency vehicle priority.

In special cases, it should override normal flow rules to ensure safety and efficiency.
