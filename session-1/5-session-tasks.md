# Challenge 1: Palindrome Checker

## Scenario

A palindrome is a word, number, or phrase that reads the same forwards and backwards.

Your task is to build a simple program that checks whether a given string or number is a palindrome.

### Requirements

Store a value that represents the input.

Your program should:

- Compare the characters from both ends of the string.
- Determine whether the value is a palindrome.
- Print an appropriate message indicating the result.

### Expected Output

```
Palindrome
```

or

```
Not Palindrome
```

# Challenge 2: Length of Last Word

## Scenario

Text editors and search engines often need to analyze user input.

Your task is to determine the length of the last word in a sentence.

### Requirements

- Store a sentence.
- Ignore any leading or trailing spaces.
- Find the last word.
- Print the length of the last word.

### Expected Output

```
Length of Last Word: 5
```

# Challenge 3: Restaurant Ordering System

## Scenario

Create a restaurant billing application used by a cashier.

The system should calculate the customer's final bill based on meal selections and additional services.

### Requirements

Store:

- Customer Name
- Meal Name
- Quantity
- Meal Size
- Drink Option
- Delivery Option

### Business Rules

- Different meal sizes have different additional costs.
- Drinks increase the total price.
- Delivery has an extra charge.
- Apply discounts for large orders.
- Remove unnecessary spaces from the customer's name before displaying it.

### Expected Output

Display a professional receipt containing:

- Customer Name
- Order Details
- Subtotal
- Delivery Fee
- Discount
- Final Total

# Challenge 5: Smart Electricity Billing System

## Scenario

Build an electricity billing application for a utility company.

### Requirements

Store:

- Customer Name

# ⭐ Final Challenge: Online Food Delivery System

## Scenario

Develop the checkout system for a food delivery application similar to Uber Eats or Talabat.

### Requirements

Store:

- Customer Name
- Restaurant Name
- Meal Name
- Quantity
- Price
- Delivery Distance
- Coupon Code
- Payment Method
- Premium Membership Status

### Business Rules

Your application should:

- Calculate the subtotal.
- Calculate delivery fees based on distance.
- Offer free delivery for premium users.
- Validate coupon codes and apply discounts.
- Apply service fees based on the payment method.
- Calculate taxes.
- Calculate the final total.
- Display a complete order summary.

### Expected Output

Display a complete invoice including:

- Customer Information
- Restaurant
- Order Details
- Delivery Fee
- Discounts
- Taxes
- Service Fees
- Final Total
- Payment Method
